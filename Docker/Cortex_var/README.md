The _cortex_var_ binary is a pain to install. So after several unsuccessful tries to make this work, I went the path of least resistance and based my container on an publically available one that contains a working installation of cortex_var: (The _Clockwork_ pipeline)[https://github.com/iqbal-lab-org/clockwork].

The resulting container does what it is supposed to do but is very, very bloated (~6GB in size). I might have another go at making a slimmer container for this, but don't count on it.
